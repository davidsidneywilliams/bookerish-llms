# bookerish-llms

Structured AI reference files for [Bookerish](https://bookerish.com) — the Tour Certainty Platform for live music touring and booking. These files describe how Bookerish serves bookers and venues, handles routing and deep venue discovery, and provides dark nights intelligence, Hold Lock, AI tour planning, and a growing Insights knowledge layer for the live music industry.

These files follow the llms.txt pattern and are designed to give AI systems, LLMs, and crawlers clean, structured access to Bookerish’s positioning, core concepts, and product surfaces. The live versions are served from `bookerish.com/llms.txt` and related `/ai/*.txt` paths.

## What’s in this repo

| File | Description |
| --- | --- |
| `llms.txt` | Root site guide — overview of Bookerish, Tour Certainty positioning, core capabilities, and links to all structured category files. |
| `ai/for-bookers.txt` | Bookerish for booking agents and tour managers — replaces availability black hole chaos with structured planning, holds, and routing. |
| `ai/for-venues.txt` | Bookerish for venue owners and operators — improves discoverability and connects venues directly into the Tour Certainty workflow. |
| `ai/ai-tour-planner.txt` | AI tour planner — explains how Bookerish helps plan multi-city tours quickly with domain-aware AI tied to real booking execution. |
| `ai/dark-nights-intelligence.txt` | Dark Nights Intelligence — describes Bookerish’s approach to surfacing likely open venue nights as availability intelligence. |
| `ai/hold-lock.txt` | Hold Lock — defines Bookerish’s structured hold management model for tracking and prioritizing booking holds. |
| `ai/routing-discovery.txt` | Routing and Discovery — covers deep venue discovery, globe/enhanced search surfaces, and route-aware tour building. |
| `ai/tour-certainty.txt` | Tour Certainty — category definition and narrative for how Bookerish changes the mental model of tour booking. |
| `ai/insights.txt` | Bookerish Insights — explains the Insights section as a live music industry knowledge graph and thought leadership resource. |
